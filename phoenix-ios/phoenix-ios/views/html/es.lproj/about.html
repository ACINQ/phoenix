Phoenix es una billetera Bitcoin que usa la red de Lightning para enviar y recibir pagos.

Es un software gratuito de código abierto, desarrollado por ACINQ bajo la licencia Apache 2.0 License.

Tenga en cuenta que esta billetera no tiene custodia: usted tiene la custodia exclusiva de la semilla de 12 palabras de la billetera. No le dé esta semilla a nadie y tenga cuidado con el fraude.

Si tiene preguntas, consulte las preguntas frecuentes. También puede ponerse en contacto con soporte técnico para obtener ayuda.

Los tipos de cambio de Bitcoin / Fiat se obtienen de varias API de terceros:

  • Blockchain.info
  • Coinbase
  • Desarrollado por Coindesk
  • Bluelytics
  • Yadio
